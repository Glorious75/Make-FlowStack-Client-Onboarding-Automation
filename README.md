## 🚀 Project Overview

FlowStack Solutions needed a way to onboard new clients professionally and instantly — without any manual administrative work. Before this system, every new client required staff to manually create CRM records, set up file storage, send welcome emails and notify the internal team — a process taking **2 to 3 hours per client**.

This project delivered a **fully automated 6-step onboarding pipeline** that handles everything from the moment a client submits a form to the moment the team is notified — in under 2 minutes, 24 hours a day, 7 days a week.

## 🔴 The Problem

Before automation, the FlowStack team was spending **2 to 3 hours manually onboarding every new client:**

- Manually creating CRM records in Airtable
- Setting up Google Drive folders for each client
- Writing and sending personalised welcome emails
- Notifying the internal team via Slack
- Verifying data accuracy across all systems

---

## ✅ The Solution: Complete Automation Workflow

A **7-step sequential workflow** built in Make.com, triggered automatically every 15 minutes. All modules confirmed active and running in production.

---

### 🔵 Step 1 — Make.com Workflow

![Make.com Workflow](https://raw.githubusercontent.com/Glorious75/Make-FlowStack-Client-Onboarding-Automation/main/Flowstack%20Solutions%20%20complete%20workflow.jpg)

*Complete 7-module automation workflow in Make.com — all modules active and confirmed running*

| Step | Module | Tool | Function |
|------|--------|------|----------|
| 1 | Watch New Form Submissions | Google Forms | Triggers on new client submission |
| 2 | Create Client Record | Airtable | Creates CRM record with all client data |
| 3 | Create Google Drive Folder | Google Drive | Generates dedicated client workspace |
| 4 | Update Airtable with Drive URL | Airtable | Links Drive folder to CRM record |
| 5 | Send Welcome Email | Gmail | Delivers branded welcome email to client |
| 6 | Notify Team | Slack | Alerts internal team of new onboarding |

---

### 🟡 Step 2 — CRM Integration: Airtable

![Airtable CRM](https://raw.githubusercontent.com/Glorious75/Make-FlowStack-Client-Onboarding-Automation/main/AIRTABLE%20CRM-Updated.jpg)

Every form submission automatically creates a structured CRM record capturing the client's name, email, company, project type, start date, acquisition source, status and Drive folder URL — **all without a single manual keystroke.**
